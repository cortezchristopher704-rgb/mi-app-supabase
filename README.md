# Proyecto Supabase - Práctica

## Descripción
Este proyecto consiste en la conexión de una aplicación en JavaScript con una base de datos en Supabase.

## Tecnologías usadas
- Node.js
- Supabase
- JavaScript
- HTML

## Funcionalidades
- Conexión a base de datos
- Consulta de datos
- Inserción de datos
- Formulario web
- Autenticación básica

## Ejecución
1. Instalar dependencias:
   npm install

2. Ejecutar:
   node index.js
